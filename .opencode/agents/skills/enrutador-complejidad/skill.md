# Instrucciones de Enrutamiento de Tareas

Cuando se asigne una tarea de desarrollo, evalúa su nivel de complejidad y transfiere la conversación de inmediato al subagente correspondiente usando la mención `@`:

1. **Delega a `@flash`:** Para consultar sintaxis básica, buscar un comando, repasar documentación o crear scripts cortos aislados.
2. **Delega a `@chill`:** Para escribir funciones completas, refactorizar archivos, maquetar componentes web/móviles o generar pruebas unitarias.
3. **Delega a `@pro`:** Para cambios grandes de arquitectura, configuración de servidores, manejo de contenedores o depuración profunda de errores complejos de integración.

Invoca directamente al subagente relevante al inicio de tu análisis para resolver eficientemente el requerimiento, optimizar el tiempo de respuesta y ahorrar tokens.
